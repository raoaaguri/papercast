Read Any PDF Aloud

Open a PDF in your browser tab and hit play — PaperCast reads every page from start to finish.

Chrome

Chrome's built-in PDF viewer is sandboxed — extensions can't read text from it directly. PaperCast works around this by fetching the PDF file's raw bytes and parsing it client-side using the bundled PDF.js library.

How to use

Open any PDF URL in Chrome → click the PaperCast icon → click Full Article. You'll see a "Reading PDF… please wait" message while it fetches and parses.

Limitations

Scanned PDFs (image-only) have no text to extract and will show an error. Password-protected PDFs also cannot be read.

Firefox

Firefox uses its own PDF.js viewer which renders each page as HTML with selectable text spans. PaperCast can read these directly — no extra library or fetch step needed.

How to use

Open any PDF in Firefox → wait for all pages to finish rendering → click the PaperCast icon → click Full Article.

Tip

If the PDF shows an empty result, scroll through the full document first to let Firefox render all pages, then try again.

Live Word-by-Word Highlighting

As each word is spoken, a subtle blue underline appears directly on the page — no DOM modifications, just a fixed-position overlay that moves to the right spot using the browser's Range API.

Full Article mode — highlights scan forward through the entire article, never backtracking.

Selection mode — highlight is scoped strictly inside the selected text range. Words outside your selection are never highlighted.

Auto-scroll — when the current word is near the edge of the viewport, the page gently scrolls to keep it visible.

Non-destructive — no changes are made to the page's DOM. The highlight is an overlay that disappears when playback stops.

Built for Technical Content

Standard TTS tools read code symbols aloud verbatim. PaperCast skips them cleanly.

Code Block Safeguards

Identifies <pre> and <code> elements instantly. Instead of spelling out symbols and brackets, inserts a clean verbal placeholder: [Code block skipped].

Math & Formula Filtering

Bypasses KaTeX, MathJax, and inline LaTeX expressions with a soft [Math expression skipped] indicator so reading velocity stays uninterrupted.

Table & Figure Summarization

Tables are replaced with a row and column count summary. Figures emit only their caption text. Neither is read cell-by-cell.

Background Playback (Chrome)

Chrome routes TTS through the background service worker. Switch tabs or minimize the window — playback continues uninterrupted without any active page connection.
